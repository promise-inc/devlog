# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in `@promise-inc/devlog`, please report it responsibly.

**Do not open a public issue.** Instead, email us at:

**security@promise.codes**

We will respond within 48 hours and work with you to resolve the issue.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |

## Scope

This policy applies to the `@promise-inc/devlog` npm package and its source code at [github.com/promise-inc/devlog](https://github.com/promise-inc/devlog).
